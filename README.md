### 🚀 DirectX 2D Dodge Game
> 상용 엔진에 의존하지 않고, C++와 DirectX API를 활용하여 그래픽스 파이프라인의 기초를 학습하기 위해 진행한 로우레벨 프로젝트입니다.

#### 🛠️ Key Learning Points
* **그래픽스 파이프라인(Graphics Pipeline) 이해:** Win32 API를 통한 윈도우 창 생성부터 디바이스 및 디바이스 컨텍스트 초기화, 스왑 체인을 통한
* 더블 버퍼링 구조를 밑바닥부터 직접 구현하며 렌더링 루프의 흐름을 파악함.
* **리소스 및 메모리 관리:** 버텍스 버퍼와 상수 버퍼를 생성하고 CPU-GPU 간의 데이터 전송 메커니즘을 이해함.
* 컴파일러가 자원을 할당하고 해제하는 과정을 관리하며 C++ 메모리 누수 방지 역량 배양.
* **수학적 연산 처리:** 정점(Vertex) 데이터와 행렬(Matrix) 연산을 통해 오브젝트의 이동 및 화면 좌표계 변환
* 프로세스의 기초적인 메커니즘을 학습함.

#### 📊 프로젝트 의의
본 프로젝트는 화려한 콘텐츠 구현보다 **상용 엔진 하단에서 동작하는 로우레벨 프레임워크의 구동 원리를 이해하는 것**에 초점을 맞추었습니다. 
이를 통해 엔진의 블랙박스 영역이었던 렌더링 연산 흐름을 시각화하고, 더 깊이 있는 C++ 최적화 코드를 작성할 수 있는 기초 체력을 다졌습니다.
